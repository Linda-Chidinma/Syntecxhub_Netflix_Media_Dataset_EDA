# Netflix EDA Project

This project analyzes Netflix content using Python and performs Exploratory Data Analysis (EDA).

## Features:
- Counts by type (Movie / TV Show)
- Yearly trends of content added
- Top genres, countries, actors, and directors
- Content growth over time
- Duration distributions
- Plots and summary report generation

## How to Run:
1. Place your dataset in the same folder as the script (or modify the path in the script)
2. Run Netflix Media dataset EDA.ipynb
3. Plots and summary report will be saved in the 'plots' folder
